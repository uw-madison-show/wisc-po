(static) createChart(chart, type, series, xAxis, yAxis, name) → {Void}

Description Creates a chart with the given parameters if specified, otherwise uses defaults for not defined variables
Parameters:
Name Type Description
chart jQueryRef jQuery reference to the object to inject the chart into
type String String of which type of chart to create - based off Highcharts chart types (Can include 'bar', 'line', 'map', 'pie').
series Array Series to plot on the chart (as defined by Highcharts at http://api.highcharts.com/highcharts#series)
xAxis Object Optional options for the xAxis (as defined by Highcharts at http://api.highcharts.com/highcharts#xAxis)
yAxis Object Optional options for the yAxis (as defined by Highcharts at http://api.highcharts.com/highcharts#yAxis)
name String Optional name for the chart (else will use default)
