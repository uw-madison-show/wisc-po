(static) convertColor(Source, Alpha, BG) → {Object}

Function to blend a rgba color onto a background rgba color. This produces an rgb color at the end. It is very useful to determine the colors for a heatmap with custom reigons.
Parameters:
Name Type Description
Source Array An array containing an rgba color
Alpha Number Alpha for the source color
BG Array An array containing an rgba color (alpha = 1)
